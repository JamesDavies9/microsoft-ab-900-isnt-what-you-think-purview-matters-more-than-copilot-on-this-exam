<h1>Why Microsoft AB-900 Exam Isn&#39;t the Copilot Exam Most People Expect It to Be</h1>

<h2>The Real Problem: You&#39;re Studying Copilot Features, But the Exam Is Mostly About Governance</h2>

<p>AB-900 (Microsoft 365 Certified: Copilot and Agent Administration Fundamentals) replaced MS-900 after it retired on March 31, 2026. Most candidates expect a Copilot features exam, but Data Protection and Governance makes up 35 to 40 percent of it, the heaviest domain, and leans heavily on Microsoft Purview rather than Copilot itself. If you prepare like it&#39;s an AI features exam, you&#39;re preparing for the wrong exam.</p>

<hr />
<h2>I Know Copilot Well, So Why Did the Microsoft AB-900 Exam Feel So Unfamiliar?</h2>

<p>Using Copilot and administering, securing, and governing it are two different skills, and AB-900 tests the second one almost entirely.</p>

<ul>
	<li><strong>Purview knowledge, not Copilot usage.</strong> Know DSPM for AI: identifying sensitive data, applying sensitivity labels, and monitoring risky AI activity through Insider Risk Management.</li>
	<li><strong>How Copilot &quot;sees&quot; data</strong>, meaning how it uses Microsoft Graph while respecting existing permissions.</li>
	<li><strong>Which admin center owns which task</strong>, across Microsoft 365, Entra, Purview Compliance Portal, SharePoint, and Power Platform admin centers.</li>
	<li><strong>Agent administration through Copilot Studio</strong>, separate from just using agents as an end user.</li>
</ul>
